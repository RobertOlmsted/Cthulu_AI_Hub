# Cthulu_AI_Hub
C'Thulu is a modular multi-agent AI system designed to coordinate various AI capabilities: Image generation, Coding assistance, TTS, Chat bot, Copy editing, embedings, LORA creation, and model tuning within a unified architecture. 
It serves as an AI platform organizing the various AI agents though shared hardware, a development enviorment for experimenting with workflows, multimodal integrations, and agentic behavior, and a foundation for applications such as Naka Radio.
